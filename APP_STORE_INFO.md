## Description

**S3 Explorer** is a powerful, native macOS app for browsing and managing your Amazon S3 buckets with ease.

**KEY FEATURES**

• **Multi-Tab Browsing** - Work with up to 5 tabs simultaneously, each with independent navigation
• **Fast File Browser** - Navigate your S3 buckets with a familiar, intuitive interface
• **File Preview** - Preview images, text files, PDFs, and more without downloading
• **Bulk Downloads** - Select and download multiple files and folders at once
• **Smart Search** - Quickly filter and find files within your buckets
• **Bookmarks** - Save frequently accessed folders for quick access
• **Multiple Connections** - Save and switch between different AWS accounts

**SECURITY FIRST**

• All credentials stored securely in macOS Keychain
• Direct connection to AWS - no intermediary servers
• Your data never passes through third-party services

**DESIGNED FOR macOS**

• Native SwiftUI interface optimized for Mac
• Dark mode support
• Keyboard shortcuts for power users
• Drag and drop support

Perfect for developers, DevOps engineers, data analysts, and anyone who works with Amazon S3 storage.
